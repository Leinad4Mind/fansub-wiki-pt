Uma vez escolhida a animação para legendar, é necessário procurar o melhor material de imagem, som e legendas para o projecto. 

A distribuição de animação no mercado é feita através de DVD, Blu-Ray e/ou streaming. Consoante o tipo de animação, o ano e as regiões de lançamentos oficiais, é necessário escolher a fonte com melhor qualidade. 

O ideal é procurar cópias na integra de DVDs (ISO), Blu-ray (BMDV ou remux) ou streaming (WEB-DL). A partir deste material base é possível fazer encode para filtrar a imagem e som, reduzir tamanho e converter para o formato mkv. Em alternativa, utilizar como base o vídeo de uma release que já tenha uma óptima qualidade, de preferência BDRip ou WEB-DL/WEBRip. 

Recomenda-se consultar a página do AniDB da respectiva animação para verificar qual a release com melhor pontuação e, consequentemente, qualidade. É oportuno saber se alguma fansub portuguesa já trabalhou nessa animação para evitar trabalhar nalgo que já foi lançado, consultem também o RevAniDB. O próximo passo é ver na página dessa fansub onde é possível descarregar. 

Extrair Legendas

A maior parte das vezes é necessário extrair legendas e/ou recursos de outras releases para iniciar o projecto de fansub. As legendas podem estar softsub (anexadas ao vídeo) ou hardsub (embutidas no vídeo). 

Legendas softsub

Se estiverem softsub, são mais simples de extrair e existem 2 opções conforme a quantidade de episódios de onde precisemos extrair. É necessário ter o MKVToolNix instalado.

Método Simples (MKVExtractGUI-2)

Quando apenas é necessário extrair faixas [legendas, ficheiro de capítulos, áudio(s)] de um ficheiro .mkv, este é o método mais rápido.

Temos apenas de abrir o MkvextractGUI-2:

1) Definir a directoria onde está instalado o MKVToolNix (apenas é necessário configurar uma vez);

2) Arrastar o ficheiro .mkv para a janela do programa;

3) Seleccionar a opção “Use Source” para salvar as faixas extraídas na mesma pasta do ficheiro .mkv;

4) E por fim, carregar em “Extract”.

Método Múltiplos Ficheiros (MKVcleaver)

Quando temos um grande número de ficheiros com uma estrutura idêntica, este método é o indicado para a extracção de faixas em volume. 

Neste caso, recorremos ao MKVclever:

1) Definir a directoria onde está instalado o MKVToolNix (apenas é necessário configurar uma vez);

2) Arrastar os ficheiros .mkv para a janela do programa;

3) Seleccionar a opção pretendida para extrair as faixas em lote. Na figura seguinte, escolheu-se a opção “Subtitle Track 1” para extrair as legendas de todos os ficheiros .mkv adicionados;

4) E por fim, carregar em “Extract Tracks!”.

Legendas harsub

Se estiverem hardsub, é necessário uma abordagem diferente para extraí-las através de OCR (Optical Character Recognition). 

Método de Legendas Bitmap 

Este tipo de legendas é típico de DVDs e disco Blu-ray, cujo formato pode ser SUP (Subtitle Picture), PGS (Presentation Grapic Stream) ou SUB/IDX (VobSub Subtitle File/Subtitle Index).

Recorremos ao Subtitle Edit que nos auxiliará a fazer o OCR automático deste tipo de legendas. 

1) Clicar em "Ficheiro" → “Abrir…” para seleccionar o ficheiro de legendas;

2) Confirmar a instalação do Tesseract OCR mais recente (5.5.0). E transferir o idioma das legendas abertas anteriormente;

3) Selecionar o Método de ORC Tesseract 5.5.0, as definições seleccionadas na figura seguinte e clicar em “Iniciar OCR”;

4) Consoante o tamanho das legendas, este processo pode demorar alguns minutos. No final, será gerado uma legenda com os tempos e respectivas linhas;

5) Resta agora, clicar em "Ficheiro" → “Guardar como…” para posteriormente editarmos as legendas no Aegisub;
