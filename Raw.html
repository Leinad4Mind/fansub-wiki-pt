Uma vez escolhida a animação para legendar, é necessário procurar o melhor material de imagem, som e legendas para o projecto. 

A distribuição de animação no mercado é feita através de DVD, Blu-Ray e/ou streaming. Consoante o tipo de animação, o ano, e os tipos e regiões de lançamentos oficiais, é necessário escolher a fonte que tenha melhor qualidade. 

O ideal é procurar cópias na integra de DVDs (ISO), Blu-ray (BMDV ou remux) ou streaming (WEB-DL). A partir deste material base é possível fazer encode para filtrar a imagem e som, reduzir tamanho e converter para o formato mkv. Outra opção, é utilizar como base o vídeo de uma release que já tenha uma óptica qualidade. 

É recomendado consultar a página do AniDB da respectiva animação para verificar qual a release com melhor pontuação e, consequentemente, qualidade. Torna-se oportuno saber se alguma fansub portuguesa já trabalhou nessa animação. O próximo passo é ver na página dessa fansub onde é possível descarregar. 

Extrair Legendas:

A maior parte das vezes é necessário extrair legendas e/ou recursos de outras releases para iniciar o projecto de fansub. As legendas podem ser softsub (anexadas ao vídeo) ou hardsub (embutidas no vídeo). 

Se estiverem softsub, são mais simples de extrair e existem 3 opções conforme a quantidade de episódios de onde precisemos extrair.
