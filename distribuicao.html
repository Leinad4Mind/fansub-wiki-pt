Multiplexagem do Projecto

Nesta fase, vamos juntar todas as faixas necessárias (vídeo, áudio(s), legendas, ficheiro de capítulos e fonts utilizadas) num só ficheiro .mkv para poder ser partilhado e visualizado posteriormente.

 

 

 

 

Nomear o Vídeo

A típica nomenclatura para designar os ficheiros deve conter a seguinte informação:

 

Nome do anime;

Ano quando existem múltiplas versões. Por exemplo: Hunter x Hunter (1999) e Hunter x Hunter (2011). Colocar sempre nos filmes, OVAs ou especiais;

Informação da fonte, ou seja, BDRip, WEB-DL, WEBRip ou DVDRip; No caso de WEB-DL ou WEBRip, colocar também AMZN para Amazon, DSNP para Disney+, e NF para Netflix. 

Resolução juntamente com o tipo de varrimento, p para progressivo e o para interlaçado, ou seja, 1080p, 720p, 576p ou 480i;

 

 

 

 

CRC (Cyclic Redundancy Check) 

O CRC é um número de verificação que certifica se uma cópia de um ficheiro é igual à sua origem ou fonte.

Serve para verificar se os ficheiros são os correctos ou correspondem à última versão lançada pela fansub, como tal todas os lançamentos devem estar acompanhadas com CRC. 

Para adicionar o CRC a um ficheiro podemos utilizar tanto o RapidCRC como o RapidCRC Unicode. Basta a abrir o programa, arrastar o ficheiro para a janela, que a verificação começa automaticamente. Caso
