Criação de Capítulos

O método mais prático e simples de criar capítulos é através de um ficheiro bloco de notas em formato .txt.

Método Manual

A forma mais directa consiste em criar um documento de texto .txt com o Bloco de Notas.

A sintaxe de cada capítulo deve indicar o respectivo número (por ordem sequencial), o tempo e o nome.

CHAPTER##=HH:MM:SS.MS

CHAPTER##NAME=NOME_DO_CAPÍTULO

De seguida, apresenta-se um modelo típico de capítulos para um episódio. Podem adicionar ou remover os capítulos que desejarem, porém devem respeitar a formatação:

CHAPTER01=00:00:00,000
CHAPTER01NAME=Abertura
CHAPTER02=00:01:48.08
CHAPTER02NAME=Prólogo
CHAPTER03=00:02:59.07
CHAPTER03NAME=1.ª Parte
CHAPTER04=00:10:44.79
CHAPTER04NAME=Intervalo
CHAPTER05=00:10:55.13
CHAPTER05NAME=2.ª Parte
CHAPTER06=00:22:31.83
CHAPTER06NAME=Encerramento
CHAPTER07=00:24:07.92
CHAPTER07NAME=Antevisão

Método Automático

Caso pretendam agilizar o processo de criar o ficheiro de capítulos, podem usar o "Plug-in Aegisub - Gerador de Capítulos" que se encontra na página Recursos.

As configurações recomendadas ("Automatização" → "Gerador de Capítulos" → "Opções" → "Configuração do Plug-in") para este plug-in são as seguintes:

1) Criar linha com o nome do capítulo;

2) Ajustar o tempo inicial da linha ao frame pretendido do início do capítulo;

3) Escrever “chapter” (sem distinção entre maiúsculas e minúsculas) no campo “Efeito”;

4) Colocar a linha em “comentário”;

5) Repetir os últimos passos consoante o número de capítulos que pretender;

 

6) Para gerar o ficheiro de capítulos clicar em “Automatização" → "Gerador de Capítulos" → "Exportar" → "Exportar Como…” e escolher a directoria para guardar.

Nomear o Vídeo

A típica nomenclatura para designar os ficheiros deve conter a seguinte informação:

  • Nome do grupo de fansub ou freelancer;
  • Nome do anime;
  • Ano quando existem múltiplas versões. Por exemplo: Hunter x Hunter (1999) e Hunter x Hunter (2011). Colocar sempre nos filmes, OVAs ou especiais;
  • Nas séries pode-se colocar o número de temporada e episódio no formato "SXXEYY”, por exemplo S02E15.
  • Informação da fonte, ou seja, BDRip, WEB-DL, WEBRip ou DVDRip; No caso de WEB-DL ou WEBRip, colocar também AMZN para Amazon, DSNP para Disney+, e NF para Netflix.
  • Resolução juntamente com o tipo de varrimento, "p" para progressivo e “i" para interlaçado, ou seja, 1080p, 720p, 576p ou 480i;
  • CRC.

Exemplos:

  • [n3-Anime] Death Note - 01 [BDRip][720p][F988D639].mkv
  • Attack.on.Titan.S04E29.1080p.BluRay.x265-n3Anime.mkv
  • [MangAnime]Cossette_no_Shouzou_-_OVA_01v2_[DVD][EC20153D].mkv
  • Shin Tennis no Ouji-sama: Hyotei VS Rikkai - Game of Future - OVA 1 [BDRip.1080p][Key][CRC].mkv

Multiplexagem do Projecto

Nesta fase, vamos juntar todas as faixas necessárias (vídeo, áudio(s), legendas, ficheiro de capítulos e tipos de letra utilizadas) num só ficheiro .mkv para poder ser partilhado e visualizado posteriormente.

Para fazer a multiplexagem do projecto recorremos ao MKVToolNix GUI:

1) Arrastar o vídeo, legendas e áudios para este campo;

2) Personalizar a ordem de apresentação das faixas.

3) Definir as opções de cada faixa (nome, idioma, Default, etc) 

4) Colocar o título do ficheiro;

5) Anexar o ficheiro .txt criado previamente com os capítulos. Definir o idioma para “Portuguese” e o “Character Set” para "UTF-8";

6) Anexar os tipos de letra utilizados nas legendas;

7) Colocar o nome do ficheiro, utilizando a nomenclatura mencionada anteriormente;

8) Iniciar a multiplexagem!

 

 

Recomenda-se gravar as configurações de multiplexagem de cada ficheiro na mesma directoria onde estão as legendas, o vídeo, o ficheiro de capítulos e os tipos de letra. No futuro, caso se realize alguma correcção às legendas e seja necessário fazer a multiplexagem, basta carregar estas configurações e clicar em “Start Multiplexing”.

 

CRC (Cyclic Redundancy Check) 

O CRC é um número de verificação que certifica se uma cópia de um ficheiro é igual à sua origem ou fonte.

Serve para verificar se os ficheiros são os correctos ou correspondem à última versão lançada pela fansub, como tal todas os lançamentos devem estar acompanhadas com CRC. 

Para adicionar o CRC a um ficheiro podemos utilizar tanto o RapidCRC como o RapidCRC Unicode. Basta a abrir o programa, arrastar o(s) ficheiro(s) para a janela, que a verificação começa automaticamente, e escolher a opção de adicionar o CRC ao(s) ficheiro(s).
