A seguinte lista de tags (etiquetas) suportada pelo formato de legendas ASS (Advanced Substation Alpha).

 

Estrutura Geral

 

Caracteres Especiais

As seguintes tags são escrita no meio do texto, e não dentro de blocos de chavetas { }.

 

Quebra de Linha

\N - Insere uma quebra de linha forçada, independentemente do estilo de quebra automática. Atenção que é um “N” maísculo.

Tags Básicos

 

Negrito

\b2 - Desactivar negrito

 

Itálico

\i1 - Aplicar itálico

\i2 - Desactivar itálico

 

Sublinhado

\u1 - Aplicar sublinhado

\u2 - Desactivar sublinhado

 

Rasurado

\s1 - Aplicar rasurado

\s2 - Desactivar rasurado

 

Tamanho de Letra

\fs<tamanho> - Define o tamanho da letra em pixéis.

Para redefinir o tamanho de letra para o tamanho padrão do estilo da legenda, utiliza apenas {\fs} ou {\fs0}.

\fn<nome> - Define o tipo de letra.
